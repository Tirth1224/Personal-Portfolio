## Setup and Installation
1. Clone the repository:
    ```sh
    git clone https://github.com/TirthPatel/personal-portfolio.git
    cd personal-portfolio
    ```
2. Install the dependencies:
    ```sh
    npm install
    ```
3. Run the development server:
    ```sh
    npm start
    ```
